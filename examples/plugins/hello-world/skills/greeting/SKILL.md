# Greeting Guide

A skill for crafting effective, personalized greetings.

## Usage

Use the `hello-world_greet` tool to generate a greeting:

```
hello-world_greet(name: "Alice")
```

## Greeting Styles

- **friendly**: Warm, casual greeting (default)
- **formal**: Professional, respectful greeting
- **enthusiastic**: Energetic, excited greeting

## Tips

- Always use the person's name when available
- Match the greeting style to the context (formal for business, friendly for casual)
- Keep greetings brief but genuine
